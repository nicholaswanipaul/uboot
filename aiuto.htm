Istruzioni

Il programma una volta caricato presentera' una maschera attraverso la quale e' possibile effettuare le ricerche. Il tempo medio di caricamento e' di circa 30 secondi ma su computer particolarmente vecchi o lenti potrebbe impiegare molto piu' tempo. Si consiglia su questo tipo di computer di eseguire il programma in modalita' nativa.

Windows

Il programma dovrebbe partire automaticamente appena inserito il cd, se cosi' non fosse sara' necessario accedere attraverso le Risorse del computer al lettore cd nel quale si trova il programma e fare doppio click sul file windows.bat. Con Windows 95 il programma viene eseguito in modalita' nativa e sara' percio' necessario confermare la richiesta in tal senso. Il programma e' compatibile con tutte le versioni di windows con almeno 64Mb di memoria a partire da Windows 95.

Linux

Il programma dovrebbe partire automaticamente appena inserito il cd, se cosi' non fosse sara' necessario eseguire il file linux.sh all'interno del cd stesso. Alcune distribuzioni non permettono l'esecuzione dei programmi da supporti rimovibili, in questo caso sara' necessario eseguire da console il comando: 

 sudo mount -o remount,exec /cdrom/ ; /cdrom/linux.sh
dove /cdrom/ e' la directory in cui e' montato il cd. Il programma dovrebbe funzionare con tutte le distribuzioni di linux con almeno 64Mb di memoria. Se cosi' non fosse verificare che le dipendenze del file eja.it/qemu.bin siano tutte soddisfate.

Mac OS X

E' sufficiente aprire il file macosx.command dal cd. Il programma dovrebbe funzionare per le versioni di Mac OS X a partire dal 10.3

Modalita' nativa

In caso di problemi nel caricamento del programma, con computer molto vecchi o con altri sistemi operativi su piattaforma x86 e' possibile usare il programma riavviando il computer dal cd (normalmente e' sufficiente lasciare il cd nel lettore primario e riavviare il computer). In questo caso la tastiera sara' impostata automaticamente come italiana standard e per uscire dal programma sara' necessario cliccare sul comando Chiudi dalla pagina di ricerca del browser. Il tempo di caricamento dovrebbe essere di circa un minuto al termine del quale apparira' la maschera di ricerca.
